# Sugar Mixing System Automation

## Project Overview
This project contains the design and simulation of a basic sugar mixing automation system developed using Siemens TIA Portal V18. The project was created as a learning and practice project for PLC and HMI based industrial automation.

The system includes PLC control logic and an HMI interface for monitoring and controlling the mixing process.

> Note: This project was developed for educational and simulation purposes only and was not implemented in a real industrial environment.

## System Functions
- Sugar mixing process control
- Start/Stop operation control
- Process status visualization

## Features
- PLC logic developed in TIA Portal V18
- HMI screen design and process control
- Industrial automation simulation
- Operator-friendly monitoring interface
- Basic process automation workflow
